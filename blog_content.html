下文使用的Python环境为2.7.12

in的使用

  • in在python中用得很多，不管set，list，dict，tuple都可以使用in来判断元素是否存在
>>> a = [1,2,3,4,5]
>>> 1 in a
True
>>> 6 in a
False
  • 这里我有一个特别有意思的用法，就是我们如果设置一个输入方法，但是需要限制用户输入的内容，比如我们只允许用户输入1，2，3，4，5中的一个数，输入别的数就需要重新输入，使用in判断是否在范围内，不符合就重新输入，逻辑就变得很简单

for的用法

  • python中的for和C语言有很大不同，因为它是采用一种遍历的思想，因此不是C语言中的(;;)形式
>>> for i in a:
...     print i
1
2
3
4
5
  • 这种遍历方式可以对list，tuple，dict，set都支持，因为它们都支持in，但是对于dict有点不一样
>>> a = {'A':11,'B':22,'C':33,'D':44,'E':55}
>>> for i in a:
...     print i
A
C
B
E
D
  • 遍历dict的时候，发现in方法取到的值只有key，并没有取到value值，可以通过这种方式实现
>>> a = {'A':11,'B':22,'C':33,'D':44,'E':55}

#方法一：
>>> for i in a:
...     print i,':',a[i]
A : 11
C : 33
B : 22
E : 55
D : 44

#方法2：
>>> for i,j in a.items():
...     print i,j
...
A 11
C 33
B 22
E 55
D 44

if的使用

  • python的if也没有括号，只有:号,同时else也是这样
>>> a = 49
>>> if a > 30:
...     print 'a大于30'
··· elif a == 30:
...     print 'a等于30'
... else:
...     print 'a小于30'
a大于30

while

>>> a = 1
>>> b = 10
>>> while a<b:
...     print a

函数的用法

  • 首先python肯定有很多内置的函数，当我们不知道具体函数的具体用法时，可以使用help()来查看此函数的帮助文档，并且自定义函数也是可以的

比较有用的函数有

1.abs求绝对值

>>> abs(10)
10
>>> abs(-10)
10
>>> abs(-3.14)
3.14

2.cmp比较函数（可以理解为第一数减去第二个数，然后判断正负）

>>> cmp(1, 2)
-1
>>> cmp(2, 1)
1
>>> cmp(1, 2)
0

3.str转换成字符串的函数

>>> print '1+1='+2
TypeError: cannot concatenate 'str' and 'int' objects
>>> print '1+1='+str(2)
1+1=2

自己定义函数

>>> def abs_user(x):
··· if x >= 0:
···     return x
··· else:
···     return –x
  • 这就是一个自己的函数，和其他语言差不多，只有一个def声明的区别
  • 需要注意的是return None，return，不写return，都是返回None

返回多值

>>> def beside(x):
...     a = x-1
...     b = x+1
...     return a,b
...
>>> print beside(5)
(4, 6)
  • python支持函数同时返回多个值,其实就是通过系统将返回的多值都放在一个tuple里返回的结果

递归函数

  • 递归函数这个问题，前期没有编程思想个人感觉光用眼睛看是没有意义的
  • 需要在纸上做一部分数学运算，然后使用，比如：
fun(n) = n! 阶乘函数
fun(n) = 1 * 2 * 3 * ... * (n-1) * n = (n-1)! * n = fact(n-1) * n
fun(1) = 1

>>> def fun(n):
...     if n==1:
...             return 1
...     return n*fun(n-1)
  • 使用递归函数需要注意防止栈溢出。栈（stack）这种数据结构，每当进入一个函数调用，栈就会加一层栈帧，每当函数返回，栈就会减一层栈帧。栈的大小不是无限的，递归调用的次数过多，会导致栈溢出。 
>>> fun(10)
3628800
>>> fun(100)
93326215443944152681699238856266700490715968264381621468592963895217599993229915608941463976156518286253697920827223758251185210916864000000000000000000000000L
>>> fun(1000)
RuntimeError: maximum recursion depth exceeded

默认参数

>>> def pow_user(x,n=2):
...     s = 1
...     while n > 0:
...             n = n-1
...             s = s*x
...     return s
>>> pow_user(2)
4
>>> pow_user(2,3)
8
  • 很多函数其实有多个参数，但是使用的时候并不一定要写上所有参数，因为有些函数具有默认参数
  • 就像上面这个例子一样，可以在定义函数的时候给一个默认参数，那么调用函数的时候，如果没有指定的参数传入，就会使用默认的参数

OK:

def fun(a, b=1, c=2):
    pass

Error:

def fun(a=1, b):
    pass

不定参数个数

  • 如果不知道参数的个数，在python中也是可以实现的
>>> def fun(*a):
...     print a
...
>>> fun(1,2,3,4,5,6)
(1, 2, 3, 4, 5, 6)
  • 实际上，也可以很容易看出，其实a在定义的时候，就是变成了一个tuple，这样传入的参数全部都被放在了a这个tuple内，以一个tuple的形式传入，但是实现了不定个数参数的传入

切片

list切片

>>> L = ['A','B','C','D','E','F','G','H']
>>> L[:]
['A', 'B', 'C', 'D', 'E', 'F', 'G', 'H']
>>> L[1:5]
['B', 'C', 'D', 'E']
>>> L[::2]
['A', 'C', 'E', 'G']
  • 两个冒号隔开
  • 第一个是起始位（闭区间，默认是0）
  • 第二个是末位（开区间，默认是-1）
  • 第三个是步长（默认是1）

对字符串切片

  • 很多编程语言对字符串处理有很多函数，用起来很繁琐,python直接使用切片机制来处理字符串
>>> 'Athrun'[:3]
'Ath'
>>> 'Athrun'[-3:]
'run'
>>> 'Athrun'[::2]
'Ahu'
#另外字符串本身还被赋予了方法，比如：
>>> 'abc'.upper()
'ABC'

生成列表式

>>> range(1, 10)
[1, 2, 3, 4, 5, 6, 7, 8, 9]
#很明显，这是左闭右开的

>>> [x*5 for x in range(1, 10)]
[5, 10, 15, 20, 25, 30, 35, 40, 45]
#可以很方便构造list

>>> [x*5 for x in range(1, 10) if x %2 == 0]
[10, 20, 30, 40]
#内部还可以使用if来筛选结果

>>> [x+y for x in 'AB' for y in '12']
['A1', 'A2', 'B1', 'B2']
#还可以for来多层构建
  • 上面的式子相当于
>>> L = []
>>> for x in 'AB':
...     for y in '12':
...             L.append(x+y)
>>> print L
['A1', 'A2', 'B1', 'B2']
